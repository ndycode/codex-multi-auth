# Dimension C — Auth/OAuth/Token Lifecycle Deep-Dive

PKCE verdict: probable strong. Repo uses `generatePKCE()` plus `S256`, but entropy source lives in external dependency code not audited here.

refresh-queue race verdict: verified locally. Same-token refreshes are deduplicated, rotation from old refresh token to new refresh token is tracked, and persisted refresh commits roll back in-memory state on write failure.

Port 1455 SSOT verdict: absent. Port and host are duplicated across auth, server, and UI copy/html instead of being derived from one parsed redirect constant.
