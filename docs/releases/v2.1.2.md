## v2.1.2 — install beside official codex without bin collisions

Removes the package-owned global `codex` binary so `codex-multi-auth` can be installed alongside official Codex npm, native, and Homebrew distributions.

### Fixes
- Dropped the global `codex` bin to avoid npm bin collisions with official Codex. (#458)
- `codex-multi-auth-codex` remains the explicit forwarding wrapper; `codex-multi-auth` remains the account-management command family. (#458)

### Other
- Updated install and migration docs to separate official Codex installation from this package's commands. (#458)
- Validated against package-bin, wrapper, documentation, script-lint, and packed-install smoke checks. (#458)
