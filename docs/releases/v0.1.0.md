# Release v0.1.0

Release date: 2026-02-27
Type: stable (`latest`)

## Highlights

- Stable release of Codex-first multi-account OAuth workflow.
- Canonical command family finalized on `codex auth ...`.
- Dashboard/settings/backend controls consolidated and documented.
- Reliability modules enabled for refresh, quota deferral, and failover.

## Install

```bash
npm i -g @openai/codex
npm i -g codex-multi-auth
```

## Core Commands

```bash
codex auth login
codex auth list
codex auth check
codex auth forecast --live
codex auth fix --dry-run
codex auth doctor --fix
```

## Notes

- Scoped prerelease package `@ndycode/codex-multi-auth` is legacy-only.
- New installs should use `codex-multi-auth`.
- Canonical paths are under `~/.codex/multi-auth`.

## Validation Snapshot

Release gate commands:

- `npm run lint`
- `npm run typecheck`
- `npm test`
